Dominio y hosting

El dominio es un nombre fácil de memorizar que identifica el lugar donde tenemos guardada nuestra página web. Un dominio está compuesto de dos partes separadas por un punto: el nombre y la extensión. El nombre es la primera parte del dominio ejemplo: “neritecnologia”, la extensión es la segunda parte y sirve para identificar qué tipo de dominio es, .com .es .net .org, etc. El alojamiento o hosting es el lugar donde está ubicada nuestra página web. Normalmente se trata de un pequeño espacio en el disco duro de un servidor que está conectado a Internet por lo que la información guardada en él puede ser visitada desde cualquier sitio con acceso a la red.
